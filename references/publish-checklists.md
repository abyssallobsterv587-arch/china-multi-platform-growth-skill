# Publish Checklists

Use this before handing work to a browser or before human publish review.

## Douyin / Kuaishou Video Draft

Required:
- Hook line
- Cover text
- Caption
- CTA
- Product or destination link

Check:
- First 2 seconds are clear
- No unsupported claims
- CTA is singular and simple

## Xiaohongshu Note

Required:
- Search-friendly title
- Cover text idea
- Body draft
- Tags / topics
- Comment bait line

Check:
- Feels like discovery content, not raw spam
- Uses user language, not hard-sell jargon
- Keeps claims realistic

## Ecommerce Listing

Required:
- Product title
- Main image direction
- 3 to 5 benefit bullets
- Long description
- FAQ
- After-sales / refund note

Check:
- Keywords lead the title
- Benefits are concrete
- No fake scarcity or fake sales claims

## Chat Funnel

Required:
- Opening message
- Qualifying question
- Core answer
- Next-step CTA

Check:
- No mass unsolicited messaging
- One CTA per message
- Human takeover point is defined

## Universal Final Check

Before publish:
- Link is correct
- Price is correct
- Product name is consistent
- CTA matches destination
- Human approval point is identified
