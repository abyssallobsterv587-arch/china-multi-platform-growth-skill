# Install and Quickstart

This repository is a reusable OpenClaw / Codex skill for China-focused growth operations.

It does not auto-post by default. It prepares:
- product selection
- platform-specific copy
- browser-ready field values
- publish checklists
- review workflows

## Quick Use

Tell your coding agent:

```text
Use the china-growth-ops skill. Score one product, choose one primary platform, generate platform-specific copy, prepare browser-ready field values, and stop before final publish.
```

## Local Readiness Check

Run:

```bash
bash scripts/doctor.sh
```

This checks for:
- `git`
- `gh`
- `openclaw`
- `Google Chrome`

It also prints the next recommended action depending on what is missing.

## What This Skill Is Best At

- Product-angle testing
- Douyin / Xiaohongshu / Kuaishou content prep
- Ecommerce listing copy prep
- Semi-automated browser workflows
- Daily review and iteration

## Safety Boundary

Use this as a semi-automatic operating system:
- draft and prepare automatically
- review and publish with human confirmation

Do not treat it as a default mass-posting or spam automation tool.
